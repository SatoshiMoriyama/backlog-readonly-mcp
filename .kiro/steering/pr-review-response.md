# PRレビュー対応ガイドライン

PRレビューの指摘に対しては以下の方針で対応してください：

## 基本方針

**自動修正**: 以下は判断不要で即座に修正する
- 文法・記法の誤り（マークダウン、タイポ、スペースなど）
- コーディング規約違反（命名規則、インデント、フォーマットなど）
- 明らかな間違いの修正

**ユーザーに確認**: 以下は必ずユーザーに聞く
- 仕様や設計の変更
- 技術選択の変更
- ビジネスロジックの変更
- 判断に迷うもの全て

## 対応手順

1. PRレビューコメントを確認
2. 自動修正できるものは即座に修正
3. 判断が必要なものはユーザーに質問
4. 修正完了後にコミット・プッシュ

迷ったら必ずユーザーに聞くこと。
